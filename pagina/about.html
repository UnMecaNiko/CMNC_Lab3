Dato cursioso

¿Sabías que más de la mitad del tráfico de internet no es de humanos?

Se calcula que más del 60% del tráfico de Internet no es humano. Es decir, se trata de tráfico generado por bots o robots. La mayor parte de este tráfico procede de los motores de búsqueda con un 30% del total, aunque no es bueno en todos los casos. La otra parte del tráfico de las máquinas es de bots maliciosos, como las herramientas de hacking o SPAM.

PT-ROUTER-NM-1AM

La tarjeta PT-ROUTER-NM-1AM cuenta con conectores dobles RJ-11, que se utilizan para conexiones de servicios telefónicos básicos. El WIC-1AM usa un puerto para la conexión a una línea telefónica estándar, y el otro puerto se puede conectar a un teléfono analógico básico para usar cuando el módem está inactivo.

PT-SWITCH-NM-1CE

El PT-SWITCH-NM-1CE cuenta con un solo puerto Ethernet que puede conectar una red troncal LAN que también puede admitir seis conexiones PRI para agregar líneas ISDN o 24 puertos sincrónicos/asincrónicos.

PC

El módulo WMP300N proporciona una interfaz inalámbrica de 2,4 GHz adecuada para la conexión a redes inalámbricas. El módulo admite protocolos que utilizan Ethernet para el acceso a LAN.

Punto de acceso

El PT-REPEATER-NM-1CE cuenta con un solo puerto Ethernet que puede conectar una red troncal LAN que también puede admitir seis conexiones PRI para agregar líneas ISDN o 24 puertos sincrónicos/asincrónicos.
